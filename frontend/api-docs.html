Developer API Docs

Integrate the recruitment intelligence engine of ResuMind AI directly into your platform using our RESTful endpoints.

Authentication

Currently in Public Beta. API Keys are generated automatically for registered users. Provide your key in the header of every request.

X-API-KEY: your_api_key_here
POST

/api/analyze

Upload a resume (PDF/DOCX) and a job description to receive a full AI analysis, including match scores and predictive career data.

Body Parameters (Multipart Form Data)

ParameterTypeDescription
resumeFileThe resume file (PDF or DOCX). Max 5MB.
job_descriptionStringThe text of the target job description.

Example Request (cURL)

curl -X POST "https://resumind-api.com/api/analyze" \
     -H "Content-Type: multipart/form-data" \
     -F "resume=@my_resume.pdf" \
     -F "job_description=Looking for a Python Guru..."
POST

/api/career-path

Generates a multi-stage career progression roadmap based on current role and skills.

Request Body (JSON)

{
  "current_role": "Software Developer",
  "skills": ["Python", "FastAPI", "Docker"]
}
